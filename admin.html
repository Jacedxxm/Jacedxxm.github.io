Admin — Contenuti del portfolio

Modifica qui i contenuti del sito, poi clicca "Scarica JSON" per ottenere un file data.json aggiornato. Carica quel file nella stessa cartella di portfolio.html sul tuo hosting (sovrascrivendo quello vecchio, se presente) e il sito si aggiorna automaticamente. Nessun contenuto viene inviato online da questa pagina: tutto resta nel tuo browser finché non scarichi il file.
